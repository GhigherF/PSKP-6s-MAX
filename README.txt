БЫСТРЫЙ СТАРТ:
--------------
1. start.bat          - Запустить приложение
2. stop.bat           - Остановить приложение
3. restart-docker.bat - Перезапустить

ДОСТУП:
-------
http://localhost

ЕСЛИ ПОСТЫ НЕ ВИДНЫ:
--------------------
1. Откройте браузер
2. Нажмите F12
3. Выполните в консоли:

   localStorage.removeItem('viewed_posts');
   location.reload();

ИЛИ откройте файл: clear-viewed.html
